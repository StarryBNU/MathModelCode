# MathModel

常见模型分类：

- 评价与决策类模型（用于对多个方案、系统或对象进行综合评价与排序。
	* 主观赋值法（依赖专家经验或主观判断）
		+ 层次分析法（AHP）
		+ 模糊综合评价法
		+ 德尔菲法（专家调查法） 
	* 客观赋值法（基于数据自身信息）
		+ TOPSIS（优劣解距离法）
		+ 主成分分析（PCA）
		+ 灰色关联分析
		+ 数据包络分析（DEA）
		+ 熵权法
- 预测类模型（用于对未来的趋势、数量或状态进行预测。
	* 时间序列预测
		+ 移动平均法
		+ 指数平滑法
		+ ARIMA模型
		+ 灰色预测模型（GM(1,1)）
	* 回归与机器学习预测
		+ 线性/非线性回归
		+ 时间序列回归
		+ 支持向量机（SVM）
		+ 神经网络（ANN）
		+ 随机森林、XGBoost等集成学习模型
- 优化类模型（用于在约束条件下寻找最优解或满意解。
	* 线性与非线性规划
		+ 线性规划（LP）
		+ 整数规划（IP）
		+ 非线性规划（NLP）
	* 组合优化与启发式算法
		+ 动态规划
		+ 遗传算法（GA）
		+ 模拟退火算法
		+ 蚁群算法
		+ 粒子群算法
		+ 多目标优化
		+ 多目标规划
		+ Pareto最优解求解方法
- 统计分析类模型（用于从数据中提取信息、检验假设或建立关系。
	* 假设检验与方差分析
		+ t检验
		+ ANOVA
		+ 卡方检验
	* 相关与回归分析
		+ Pearson/Spearman相关分析
		+ 多元线性回归
		+ 逻辑回归
	* 聚类与分类
		+ K-means聚类
		+ 层次聚类
		+ 判别分析
		+ 朴素贝叶斯分类
- 系统建模与仿真类模型（用于模拟复杂系统行为或动态过程。
	* 微分方程模型
	  	+ 常微分方程（ODE）模型
	  	+ 偏微分方程（PDE）模型
	  	+ 动力系统模型
	* 仿真模型
	  	+ 蒙特卡洛模拟
	  	+ 系统动力学模型
	  	+ 离散事件仿真
	  	+ 智能体建模（ABM）
- 网络与图论模型（用于研究网络结构、路径、流量等问题。
	* 最短路径算法（Dijkstra, Floyd）
	* 最小生成树
	* 网络流模型（最大流、最小割）
	* 复杂网络分析（中心性、社区发现）
